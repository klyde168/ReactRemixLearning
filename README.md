# ReactRemixLearning

## 專案描述

本專案使用Remix框架初始化，包含React和Tailwind CSS，並設計了兩欄式佈局。

## 備註

第1天：專案初始化與環境設置
第2天：導航欄與路由基礎

## 功能

- 實現了二欄式佈局，左欄為靜態導航欄，右欄為內容顯示區域。
- 導航欄包含首頁、關於、服務和聯絡連結。
- 點擊導航連結後，右欄會顯示對應的內容。
